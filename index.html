About

關於我

您好，我是 PLUS。於聯合大學工業設計系畢業後，就陸續從事設計相關工作，接觸比較多的是視覺設計。喜愛看動漫畫電影的興趣也奠定了對色彩敏銳度和故事創造性。從小察覺人事物的敏銳度讓我會以他人為出發點，同理並思考著對方需求，將能所協助的，貼心提供給對方。如我身上盡量帶著衛生紙或濕紙巾。

這些特質也在設計上得到了很多幫助。像在特思爾大宇宙時，處理著名的日系連鎖服飾品牌專案，思考輸出符合在地觀看者的視覺呈現，調整手機框架的 UI 視覺及內容更舒適讓觀看取得資訊(因日文字和繁體字呈現會有差異)。每季檔期或週年活動也透過我們設計上的視覺協助，讓銷售業績非常明顯提升。而得到客戶的感謝，忙碌挑戰後令人開心。也從中更體會到「用戶的體驗和感受」。

更因工作需常接觸網頁相關內容，主動自學網頁設計相關。很慶辛適逢參與職訓局的前端網頁開發技術課程受訓半年。積極的去學習到 HTML、CSS、JavaScript、Vue.js 及 RESTful API 等技術。過程中遇到卡關、自我質疑能力、焦慮時間。但盡力讓自己心態沉住氣，一步一步用目前所及能力去完成各項任務。結業前需獨立從無到有完成網頁專案。從覺得到底要怎麼開始？腦海出現真有辦法能生出來嗎？各式惡魔喃語折磨自己，但最後真的製作呈現出來。體會到“真的不要限制住自己的可能性”。也讓自己更“網前”些。

過往處理視覺設計相關的美感經驗與工業設計對於使用者體驗上讓產品更方便符合人性使用的薰陶下，期許這些能力相輔前端網頁。設計出更有質感，使用上更方便的網頁 UI 呈現，讓使用者體驗更加舒適便利。以人為出發點下持續累積多元的能力及發展更多不同的可能。
